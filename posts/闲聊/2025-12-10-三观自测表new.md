# 🌿 三观自测表 —— 认清自己 · 认清现实
### 一、世界观（认清世界）
| 一级分类     | 子类         | 主义/观点       | 定义/核心主张 |
|--------------|--------------|------------------|----------------|
| **世界观**   | 哲学取向     | 唯物主义        | 认为世界的本质是物质，物质决定意识 |
|              |              | 唯心主义        | 认为意识或精神是第一性的，决定物质存在 |
|              |              | 虚无主义        | 否定一切既定的真理和意义 |
|              |              | 建构主义        | 相信知识和现实是通过社会建构而形成的 |
|              |              | 二元论          | 认为物质和精神是两个独立且基本的存在 |
|              |              | 过程哲学        | 认为宇宙本质上是变化和过程的总和 |
|              |              | 不可知论        | 认为我们无法确知世界的本质或神的存在 |
|              |              | 神秘主义        | 强调神秘体验或直觉是理解真理的方式 |
|              | 宗教信仰     | 灵性主义        | 重视个人内在的精神探索而非传统宗教 |
|              |              | 有神论          | 相信一个或多个神的存在 |
|              |              | 无神论          | 否定任何神或超自然存在的存在 |
|              |              | 不可知论        | 认为神是否存在是不可知的 |
|              |              | 泛灵论          | 认为自然界的所有事物都有灵魂 |
|              |              | 混沌论          | 认为复杂系统的行为具有不可预测性 |
|              |              | 系统论          | 从整体和系统的角度看待复杂现象 |
|              |              | 演化论          | 相信生物和宇宙的发展是逐步进化的 |
|              | 科学观       | 机械决定论      | 认为一切现象都有确定的因果关系 |
|              |              | 反科学主义      | 怀疑科学的权威性或普遍适用性 |
|              |              | 科学实用主义    | 将科学视为工具，重视其实用性 |
|              | 认知方式     | 构建主义        | 认为知识是由个体或社会主动建构的 |
|              |              | 实用主义        | 判断依据是否“有用”，重视实际效果 |
|              |              | 理性主义        | 依赖逻辑推理和普遍原则来认识世界 |
|              |              | 经验主义        | 通过感官经验和观察来获取知识 |
|              |              | 直觉主义        | 依赖直觉、灵感或顿悟获取知识 |
|              |              | 怀疑主义        | 对一切知识主张保持留态度，持续质疑 |
|              | 人性假设     | 人性本善        | 相信人在本质上倾向善良和合作 |
|              |              | 人性本恶        | 认为人生来自私或倾向作恶 |
|              |              | 人性中性        | 认为人性没有固定倾向，环境决定行为 |
|              |              | 多重人性        | 人性复杂多变，善恶并存且可共生 |
|              | 他人观       | 情境决定论      | 行为主要由环境和情境触发而非内在特质 |
|              |              | 高信任          | 普遍相信他人诚实、善意和可信赖 |
|              |              | 低信任          | 对他人普遍持怀疑态度，不易相信 |
|              |              | 谨慎信任        | 在建立关系或获得验证后才给予信任 |
|              | 社会信任     | 制度信任        | 相信法律、政府、媒体等制度能保障公平 |
|              |              | 人际信任        | 更依赖熟人网络、情感纽带建立信任 |
|              |              | 系统怀疑论      | 对主流制度、媒体、专家保持高度警惕 |

### 二、价值观（认清现实）
| 一级分类     | 子类         | 主义/观点       | 定义/核心主张 |
|--------------|--------------|------------------|----------------|
| **价值观**   | 伦理取向     | 相对主义        | 认为道德判断依赖于文化背景和个人立场 |
|              |              | 功利主义        | 认为道德行为应追求最大幸福或利益总量 |
|              |              | 美德伦理        | 强调品格和美德比行为后果更重要 |
|              |              | 关系伦理        | 认为人与人之间的关系决定道德责任 |
|              |              | 后现代伦理      | 主张伦理是多元、相对和情境性的 |
|              |              | 义务论          | 认为行为的道德性取决于是否履行义务（如康德） |
|              | 社会取向     | 国家主义        | 强调国家利益高于个体或群体利益 |
|              |              | 集体主义        | 重视群体利益、责任与集体荣誉感 |
|              |              | 个人主义        | 强调个人自由、权利和自主性最重要 |
|              |              | 全球主义        | 主张超越国家边界，重视全人类利益 |
|              |              | 社群主义        | 强调地方性、归属感与共同体关系 |
|              |              | 身份政治        | 认为政治与价值应围绕群体身份（如性别、种族）展开 |
|              | 价值排序     | 物质优先        | 追求财富、实用、效率等现实成果 |
|              |              | 精神优先        | 强调精神追求、信仰、内在满足 |
|              |              | 工具主义        | 看重事物是否有助于达成目标 |
|              |              | 体验主义        | 重视主观感受与多样化人生体验 |
|              |              | 平衡型          | 认为物质与精神应协调发展 |
|              |              | 超验价值优先    | 追求超越现实的终极价值，如信仰、真理 |
|              | 行动观       | 自由主义        | 主张个人自由与权利优先，强调开放 |
|              |              | 保守主义        | 重视传统与秩序，反对激进改变 |
|              |              | 现实主义        | 以实用、现实为判断标准，不理想化 |
|              |              | 理想主义        | 坚持理想信念，追求道义与正义 |
|              |              | 激进主义        | 倾向于推动社会剧烈改革或革命 |
|              |              | 犬儒主义        | 不信任任何理想或制度，不参与公共事务 |

### 三、人生观（认清自己）
| 一级分类     | 子类         | 主义/观点       | 定义/核心主张 |
|--------------|--------------|------------------|----------------|
| **人生观**   | 生命目标     | 享乐主义        | 认为快乐是人生的最高追求 |
|              |              | 利他主义        | 认为帮助他人是人生的核心意义 |
|              |              | 自我实现        | 致力于充分实现自身潜能与价值 |
|              |              | 宿命论          | 认为命运不可更改，一切早已注定 |
|              |              | 虚无主义        | 认为人生没有固定目标或意义 |
|              |              | 成就主义        | 将达成成就、名望、影响力视为人生意义 |
|              |              | 佛系            | 主张顺其自然，不强求 |
|              |              | 社会建构主义    | 认为人生目标是社会赋予的，不是先天的 |
|              | 存在态度     | 参与主义        | 重视参与社会、文化、历史等过程本身 |
|              |              | 积极主义        | 相信人生充满意义，应积极应对挑战和困境 |
|              |              | 悲观主义        | 倾向于认为人生多苦难，未来不容乐观 |
|              |              | 存在主义        | 主张个体自由选择并为自己存在赋予意义 |
|              |              | 虚无主义        | 认为人生本质上没有意义或目的 |
|              |              | 顺应主义        | 认为应顺应命运或环境的安排，不强求改变 |
|              |              | 道法自然        | 主张顺应自然规律，不人为强为 |
|              |              | 反抗主义        | 强调在荒谬中反抗现实本身就是意义所在 |
|              |              | 大儒主义        | 对一切保持怀疑态度，拒绝理想主义和权威 |
|              | 时间观       | 历史决定论      | 相信个人命运受历史结构或趋势决定 |
|              |              | 当下主义        | 强调活在当下，把握当前时刻最重要 |
|              |              | 线性时间观      | 认为时间是有起点和终点的单向流动过程 |
|              |              | 循环时间观      | 相信历史和人生呈周期性重复 |
|              |              | 进步论          | 相信社会与人类会不断进步，未来优于过去 |
|              |              | 断裂论          | 认为历史是非连续的，由突变和转折主导 |
